Overview

What Cyanic actually does

A next-generation DEX aggregator built on Base. It scans multiple decentralized exchanges at once so a swap never settles for a worse price than the market has on offer.

Multi-DEX Routing

Aggregates Uniswap V3, Aerodrome, SushiSwap, PancakeSwap V3 and more through 0x Protocol v2 to find the route with the best net output.

Cross-Chain Bridge

Bring assets in from Ethereum, Arbitrum, Optimism and other connected chains and land on Base in the same flow — no separate bridging site required.

AI Agent

Describe a swap in plain language and a Claude-powered agent reads live routes and gas costs, then executes with your approval.

MEV-aware Routing

An optional private mempool path through 0x keeps larger swaps out of public order flow and away from sandwich bots.

XP & NFT Tiers

Every swap earns XP. Cross a threshold and mint an on-chain SVG NFT — Bronze through Diamond, fully on Base.

Leaderboard

A global ranking by score, volume and swap count across every Base trader routing through Cyanic.
